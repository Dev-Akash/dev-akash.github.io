Classroom Attendance Marking System

This project was a work based on Computer Vision (opencv) and Machine Learning (Tensorflow). Where the models were trained on very less quantity of data and balance the distribution of classification with synthetic data generated (using Keras data-generators). So that with just on click of classroom all students present in photo get there attendance marked.

Replication of the human hand motion in Real time

This project was major project in my graduation in Robotics & Automation. This project was created to replicate the human behavior in realtime, which could be helpful in many task where humans can't reach. The project was based on PoseNet model which detects the pose of human body in real time and mapped all of the angles and poses to our two armed humanoid actuators to perform that pose in real time. Some future works include learning human moves in real time.

Face Detection Using Supervised Machine Learning

This project was more about detecting feature points of face in the photograph and then sending them to machine learning model to detect the person rather than directly sending the complete photo to the model. The whole face detection pipeline was created using OpenCV.

MNIST Digit Recognition

This is was a project cum Machine Learning challenge on Kaggle platform, where the numbers have to be recognize using MNIST Dataset. In this project I used Ensemble method to reach the accuracy of 99.98% which would further be maximized by bagging and boosting methods.

Skin Cancer MNIST Recognition: HAM10000

This project was to recognize images of pigmented lesions, it was more of Machine learning challenge on Kaggle where the main challenge was to train on very few data samples. Implemented synthetic data generators to create more samples for the current data set to overcome the challenge.

Chest X-Ray Images (Pneumonia Detection)
